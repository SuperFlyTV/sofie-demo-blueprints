# Global Adlibs

Global Adlibs are Adlibs that are available to any Rundown created with this set of Blueprints. They can be accessed by opening the drawer at the bottom of the Rundown UI. Currently, the Demo Blueprints specify 5 Global Adlibs:

- Last Remote
  - Shows the last-used Remote input on-air
- Last DVE
  - Shows the last-used DVE configuration on-air
- Camera 1
  - Shows Camera 1 on-air
- Camera 2
  - Shows Camera 2 on-air
- Remote 1
  - Shows Remote 1 on-air
